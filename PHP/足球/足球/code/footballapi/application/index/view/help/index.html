让球盘
  • 预测哪一支球队在盘口指定的让球数在全场/半场/赛事某个时节赢得比赛。
  • 让球盘由交战球队、让球数（即盘口）及贴水这3个部分组成。例如曼联0.86球半1.00富勒姆。其中曼联和富勒姆就是交战球队、球半即为让球盘数、0.86/1.00即为投注双方的获胜赔率又称“贴水”。
  • 让球的算法：实际比赛的结果+让球数=竞猜结果
  • 让半球：打平算输，赢一个或以上算赢
  • 让一球：打平算输，赢一个算平，赢两个或以上算赢。
  • 以此类推。
